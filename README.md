# نرم‌افزار مدیریت رکوردهای ورود و خروج

یک برنامهٔ رومیزی سبک برای خواندن فایل رکوردهای ورود/خروج، افزودن رکورد جدید و
ذخیرهٔ آن‌ها در یک فایل جدید (بدون بازنویسی فایل اصلی).

## قابلیت‌ها

- **باز شدن با کادر «کشیدن و رها کردن»** در وسط صفحه؛ فایل را می‌گیرید و رها می‌کنید.
  (اگر کتابخانهٔ اختیاری نصب نباشد، از دکمهٔ «انتخاب فایل» استفاده کنید.)
- **خواندن فایل رکوردها** و نمایش آن‌ها در یک لیست قابل اسکرول.
- **فرم ثبت رکورد** در بالای صفحه با سه فیلد مهم و قالب سخت‌گیرانه:
  1. کد پرسنلی (دقیقاً ۶ رقم)
  2. تاریخ:
     - در حالت میلادی: `yyyy-mm-dd` (مثال `2026-06-22`)
     - در حالت شمسی: `yyyy/mm/dd` (مثال `1405/04/01`)
  3. ساعت به وقت تهران با قالب `hh:mm:ss` (مثال `07:39:23`)
- **راهنمای متنی (placeholder)** درون هر فیلد قالب مورد انتظار را نشان می‌دهد و
  با تغییر گزینهٔ «تاریخ شمسی» به‌طور خودکار به‌روز می‌شود.
- **ورود تاریخ به صورت شمسی**: با تیک زدن گزینهٔ مربوطه، تاریخ شمسی وارد می‌کنید
  و برنامه هنگام ثبت آن را به میلادی تبدیل می‌کند.
- **لیست موارد اضافه‌شده** در همین نشست، بین فرم و لیست اصلی نمایش داده می‌شود.
- **اسکرول خودکار و ریل‌تایم**: با هر ثبت، لیست اصلی به محل رکورد تازه اسکرول شده و
  آن ردیف برجسته (سبزرنگ) و انتخاب می‌شود.
- **دو دکمهٔ ذخیره** در پایین صفحه (همیشه در دید):
  - **ذخیره (Save)**: بازنویسی فایل باز شدهٔ فعلی (با پرسش تأیید).
  - **ذخیره به عنوان (Save As)**: انتخاب یک مسیر جدید؛ به‌طور پیش‌فرض کنار فایل
    اصلی و بدون بازنویسی آن.
- **دکمهٔ خروج**: به حالت اولیه (کادر کشیدن و رها کردن) برمی‌گردد؛ اگر تغییر ذخیره‌نشده
  داشته باشید، پیش از خروج هشدار می‌دهد.

## قالب فایل

هر خط شامل: کد، سپس «تاریخ و ساعت»، سپس ستون‌های اضافی که با Tab جدا شده‌اند:

```
   308590	2026-06-22 07:12:45	1	0	1	0
```

- ستون اول: کد ۶ رقمی فرد
- ستون دوم: تاریخ میلادی + ساعت به وقت تهران
- ستون‌های بعدی: اطلاعات اضافی که برای رکوردهای وارداتی حفظ می‌شود.

سه ستون مهم (کد، تاریخ، ساعت) در برنامه قابل ویرایش‌اند. رکوردهای جدیدی که خودتان
اضافه می‌کنید فقط همین سه مقدار را دارند.

## اجرا

```bash
cd attendance-tool
python3 attendance_app.py
```

### فعال‌سازی «کشیدن و رها کردن» (اختیاری)

```bash
pip install tkinterdnd2
```

اگر این کتابخانه نصب نباشد، برنامه همچنان کار می‌کند و برای انتخاب فایل روی کادر یا
دکمهٔ «انتخاب فایل» کلیک می‌کنید.

### نصب tkinter (در صورت خطای «No module named tkinter»)

- Debian/Ubuntu: `sudo apt-get install python3-tk`
- Fedora: `sudo dnf install python3-tkinter`
- macOS/Windows: معمولاً همراه نصب رسمی پایتون موجود است.

## تست منطق برنامه

منطق اصلی (تبدیل تاریخ و تجزیهٔ فایل) مستقل از رابط گرافیکی نوشته شده و قابل تست است:

```bash
cd attendance-tool
python3 test_core.py
```

## ساختار فایل‌ها

- `attendance_app.py` — رابط گرافیکی (Tkinter).
- `attendance_core.py` — منطق اصلی: تبدیل شمسی/میلادی، خواندن و نوشتن فایل.
- `test_core.py` — تست‌های منطق اصلی.
- `sample_records.txt` — نمونه فایل رکوردها برای آزمایش.
- `requirements.txt` — وابستگی‌ها (فقط tkinterdnd2 اختیاری).
